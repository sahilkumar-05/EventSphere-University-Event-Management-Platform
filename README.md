# EventSphere – University Event Management Platform

EventSphere is a centralized event management platform designed to help students discover and participate in university events with ease. The platform provides a seamless experience for browsing upcoming events, viewing event details, and registering directly through a modern and intuitive interface.

Students can securely create accounts, log in, and stay connected with academic, cultural, technical, and social events happening across campus. EventSphere simplifies event participation by bringing all university events into a single digital platform.

On the administrative side, the platform includes a dedicated dashboard that enables organizers to efficiently manage events and monitor user engagement. Administrators can create, update, and delete events, track registrations, and view platform statistics to better understand event participation.

## Features

### Student

* Secure Signup & Login
* Browse University Events
* View Event Details
* Register for Events
* Responsive and User-Friendly Interface

### Admin

* Add and Manage Events
* Update Event Information
* Delete Events
* Track User Registrations
* View Event and User Statistics
* Administrative Dashboard

## Tech Stack

* React Native
* TypeScript
* Native Components
* Html/Css


EventSphere aims to enhance campus engagement by providing a modern, efficient, and accessible solution for university event management.
